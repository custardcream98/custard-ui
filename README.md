# custard-ui

React 디자인 시스템 개발중!

[Storybook](https://custardcream98.github.io/custard-ui)으로 문서화됩니다.

## 사용하기

custard-ui는 node package로 배포되었습니다.

```shell
npm i custard-ui
# or yarn i custard-ui
```

본 라이브러리를 사용하려면 아래의 dependency가 필요합니다.

```json
{
  "@emotion/react": "^11.10.5",
  "@emotion/styled": "^11.10.5",
  "react": ">=17.0.0",
  "react-dom": ">=17.0.0",
  "react-icons": "^4.7.1"
}
```
